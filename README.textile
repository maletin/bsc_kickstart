h2. Weitere Informationen über BaseCondition

h3. Was ist BaseCondition

BaseCondition ist ein CSS-Framework das als Grundtechnologie den CSS Preprocessor Less nutzt. BaseCondition verfügt über ein ausgeklügeltes Cachingsystem. Im Ordner "site/stylesheets/" abgelegte Less-Dateien werden "OnTheFly" zu CSS-Dateien geparst. Weitere Informationen finden sich unter "GitHub/JoachimDoerr/BaseCondition":https://github.com/joachimdoerr/basecondition

h3. Systemvoraussetzungen

BaseCondition ist für den Einsatz auf Webserver konzipiert und benötigt für eine optimale Funktionalität PHP ab Version 4.3.2. BaseCondition arbeitet auch fehlerfrei mit localen Servertechnologien wie MAMP oder XAMPP. Für einen fehlerfreien Betreib müssen Lese- und Schreibrechte entsprechend korrekt vergeben werden.
